### Basic Example:

```js
<PlainLiteral subjectURI='example' predicateURI='title' />
```

### Pre-populated example

Default values can be pre-populated using the "value" property:

```js
<PlainLiteral subjectURI='example' predicateURI='title'
 value={{'@value': 'Lorem ipsum', '@language': 'en'}}/>
```
